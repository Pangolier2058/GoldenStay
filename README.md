# Автор работы и тема проекта.
Проект по дисциплине Web-программирование по теме "Бронирование гостиничного комплекса". Работу выполнил студент группы 609-31з направления "Программаня инженерия" Чинчевич Иван Сергеевич.

# Функциональность разработанного приложения Разрабатываемый сайт предназначен для бронирования номеров в гостиничном комплексе пользователями.

В функционал сайта входит:

1) Возможность авторизации/регистрации пользователя с использованием базы данных
2) Поиск доступных для бронирования номеров для каждого корпуса с помощью SQL запросов а базе данных
3) Выбор конкретного интересующего номера для бронирования из перечня, а также ознакомление с информацией о каждом номере и корпусе Создание заявки на бронирование посредством заполнения формы бронирования для выбранного пользователем номера
4) Просмотр статуса заявки в личном кабинете польователя с возможностью отмены бронирования

Были использованы следующие инструменты: СУБД MySQL, CSS-фреймворк BootStrap. Сам сайт написан на PHP.
